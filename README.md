# Micron_Level_Gap_measurement_system
##  Features

 **Image Input Options**
  - Load images from disk
  - Capture real-time images using a camera

 **Advanced Image Processing**
  - CLAHE contrast enhancement
  - Bilateral & median filtering
  - Sobel + Scharr edge detection
  - Canny-based fast mode

  **Accurate Gap Measurement**
  - Detects the vertical gap center automatically
  - Measures gap at multiple heights
  - Statistical analysis (mean, median, std, min, max)

  **Calibration Support**
  - Convert pixel measurements to millimeters
  - User-defined actual gap input

**Visualization**
  - Gap detection overlay
  - Measurement profile graph
  - Histogram distribution
  - Coordinate reference map
  - Zoomed inspection view

 **Automated Reporting**
  - Export results as:
    - Excel (.xlsx)
    - CSV (.csv)
  - Save annotated images

 **Quality Check**
  - Pass/Fail classification based on threshold (0.5 mm)

 **Fast Mode**
  - Reduced computation for quicker approximate results
